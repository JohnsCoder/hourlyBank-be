# Sistema web de Gerenciamento de projetos
Esse sistema é voltado para, ajudar pessoas a organizarem seus projetos e valores cobrados pela hora trabalhada.

## execução
instalar dependencias ada aplicação:
`npm i`
executar aplicação:
`npm run dev`
